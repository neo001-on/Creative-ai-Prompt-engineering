# Creative AI Prompt Engineering

A collection of AI-assisted screenplay and storytelling experiments exploring how prompt engineering can support creative writing, cinematic storytelling, and narrative design.

This project focuses on human-AI collaboration in generating emotionally engaging short film concepts and screenplay structures.

---

## Project Focus

- Prompt engineering
- AI-assisted storytelling
- Screenplay generation
- Character design
- Scene structuring
- Tamil narrative experimentation
- Human-AI creative workflows

---

## Example Project

### "ஒரு ரூபாயின் சத்தம்"

A socially grounded short film concept exploring poverty, dignity, emotional struggle, and survival through realistic cinematic storytelling.

Inspired by emotionally impactful Tamil cinema styles.
